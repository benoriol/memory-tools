# Polynomial regression demo (memory-graph sample workload)

A self-contained ~10-minute autonomous task to exercise the
memory-graph-mcp system end-to-end. The agent reads `CLAUDE.md`, runs
a small experimental sweep, and is instructed to record what it
learns at every step via the `memory_*` MCP tools.

Run with `bash demos/run-polyreg.sh` from the repo root.
